# Meta AI Loading Animation 🚀

A sleek, modern, and highly interactive recreation of the iconic **Meta AI Loading Animation** using HTML, CSS, and Vanilla JavaScript. This project focuses on smooth transitions, gradients, and clean code logic.

![Meta AI Preview](https://www.whitecoder.online/demos/meta_loading_animation/) 

## ✨ Features
- **Aesthetic UI:** Inspired by the latest Meta AI design.
- **Smooth Animations:** Built using CSS Keyframes for high performance.
- **Dynamic Logic:** Controlled via Vanilla JavaScript.
- **Lightweight:** No heavy libraries, just pure code.
- **Responsive:** Works perfectly on all screen sizes.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure.
- **CSS3:** Custom gradients and complex animations.
- **JavaScript (ES6):** State management and DOM manipulation.
